Como fue realizado el ejercicio

El ejercicio fue realizado en su totalidad en HTML, CSS y JavaScript

En la parte front, se utilizado Bootstrap como libreria externa para hacer un rapido diseño y una libreria para solventar los problemas que trae CORS durante el desarrollo

Dentro de la documentacion de Front se encuentra mas en detalle que funcion tiene cada archivo en el programa

Para la parte back-end, se utilizado NodeJS junto con Express, inicializado desde el Generador de Express, para poder trabajar con un modelo MVC correctamente
