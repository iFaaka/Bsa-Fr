Que archivos tengo

El front simula ser parte del MVC que tiene el BackEnd, por lo que consta de una carpeta public donde dentro de ella se separan los archivos.

Los archivos HTML

  • Index: Es el home de nuestro Front, en el contamos con una descripcion de la actividad y las redirecciones deseadas
  • Products: Aqui sucede la magia, es la pagina donde se localiza la tienda. Cuenta con un navbar identico al del index, la estructura basica para el dropdown button donde se encuentran las categorias que se extraen de la base de datos y agrega unas extras de mi autoria, como productos con descuento y recibir todos los productos. Tambien agrega el input que servira como buscador
  • DocumentacionAPI y DocumentacionFront: Son los HTML correspondientes a la doc de esta pagina.

Los Archivos CSS

  • Documentacion.css: Guarda los estilos para ambas documentaciones
  • Navbar: Comprende los estilos del navbar.
  • Title&Categories: Comprende los estilos para el titulo de la tienda y todo lo que se refiere al input buscador y el dropdown de las categorias.
  • Products: Comprende los estilos para los productos, como las cards en las que estan introducidos
  • Index: Comprende los estilos para nuestra pagina de inicio.

Los archivos JavaScript

  • createCards: Comprende toda la logica para la creacion de las tarjetas de los productos
  • getData: Comprende la logica para obtener todos los datos extraidos de la BD
  • products: Contiene las funciones que llaman a otras funciones para poder hacer consultas especificas a la API. Sera explicada en profundidad mas adelante
  • title&categories: Comprende la generacion del titulo y creacion del menu de categorias, ademas de tambien manejar el input buscador

createCards

Este archivo JS se encarga de construir las tarjetas que recibe desde la funcion en products. Para destacar, tenemos dos filtros aplicados

  • Filtrado de imagen

    En la BD hay items que no cuentan con imagenes, teniendo valores null o string varios en la propiedad url_image, por lo que filtramos las que si tienen imagen, y a las que no, les aplicamos una imagen extraida de google imagenes.

  • Filtrado de descuento

    En la BD existen items con descuento, por lo que mostraremos los dos precios, el original y el que tiene el descuento aplicado. Si existe un descuento, el precio original adquirira una clase nueva de CSS que tache su valor, y el precio con descuento agrandara su tama√±o y adquiere un color mas llamativo

getData

Este archivo de JS tiene la logica para traer la data de la BD, independientemente de que dato sea el que se pide

Para trabajarlo localmente, agrege una referencia a una API que permite evitar los problemas con CORS

products

Este archivo JS se encarga de llamar a la funcion para crear tarjetas como segundo argumento, para como primer argumento pasarle el link necesario para hacer la llamada a la API obteniendo los datos deseados

Ademas, tambien cuenta con una funcion para eliminar todas las tarjetas de preguntos y ejecuta por unica vez una llamada a la muestra de todos los productos

title&categories

Este archivo JS cuenta con varias funciones para el funcionamiento del Front
  • Crea el titulo de la pagina
  • Recibe las categorias de la BD y las mapea dentro de un dropdown, agregando tambien funciones para filtrar los items por categoria

    Ademas de agregar dos nuevas categorias, recibir todos los productos y productos con descuento

  • Maneja el buscador y filtra desde el front lo que se ingresa
